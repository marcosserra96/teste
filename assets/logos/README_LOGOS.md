# Logos do Portal Atletas Energisa

Arquivos versionados no GitHub:

- `logo-comite-colorida.png`: versão colorida, usada principalmente no informativo/report.
- `logo-comite-branca.png`: versão branca com fundo transparente, indicada para header, login e fundos escuros.
- `logo-login-branca.png`: alias da versão branca para a tela de login.
- `logo-informativo-colorida.png`: alias da versão colorida para o informativo.

Para alterar a logo no futuro, substitua os arquivos nesta pasta e publique novamente o GitHub Pages.
